### INSTALL

Clone it!

```
git clone https://github.com/ivan-iver/fly.git fly
```

Then install

```
$ make install
```

### UNINSTALL

```
$ make uninstall
```

### RUN

```
$ make build
$ cd bin
$ ./fly
```
