### INSTALL

```
$ go get -v github.com/ivan-iver/fly
```

### UNINSTALL

```
$ rm -rf $GOPATH/src/github.com/ivan-iver/fly
```

### RUN

```
$ make build
$ cd bin
$ ./fly
```
